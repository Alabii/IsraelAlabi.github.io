Cyclic Bikes Analysis (R)

This project explores bike-sharing trip data to uncover riding patterns between annual members and casual users. Using R and the tidyverse suite, I cleaned and merged raw datasets, engineered new features, and applied descriptive statistics with visualizations. The analysis reveals distinct behavioral trends such as casual riders taking longer trips and preferring electric bikes, while members ride more frequently during weekdays on classic bikes. Longest-Used Bike Trip Analysis with SQL (BigQuery)

In this project, I used SQL on Google BigQuery to identify the longest-used bike(s) in the Austin Bikeshare dataset and analyze which start stations accounted for the highest number of their trips. The analysis demonstrates skills in CTEs, window functions, ranking, aggregation, and joins, along with best practices in query structuring and readability. By combining performance-focused SQL features such as QUALIFY, DENSE_RANK, and percentage calculations, the project highlights my ability to transform raw trip data into actionable operational insights.
